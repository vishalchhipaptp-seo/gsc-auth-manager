# GSC Auth Manager

Desktop companion app for setting up Google Search Console authentication. Runs locally on your computer where Chrome works, and auto-uploads auth states to the VPS.

## Quick Start

1. Download this repo as ZIP and extract
2. Double-click `run.bat`
3. Enter your VPS URL (e.g. `https://james.weblinkbuzz.com`)
4. Enter your dashboard login email and password (same credentials you use on the website)
5. Click Connect — the app shows all GSC accounts
6. Click "Set Up" next to an account — Chrome opens for Google sign-in
7. Sign into Google and grant access — auth state auto-uploads to VPS

## Requirements

- Windows 10/11
- Internet connection

Python and all dependencies install automatically on first run via `run.bat`. No manual setup needed.
